## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Polarbearlover3864/CodingKing.git
cd CodingKing
```

2. Open the project:
   - Simply open `index.html` in your web browser
   - Or use a local server for best results:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have it installed)
npx http-server
```

3. Visit `http://localhost:8000` in your browser

## Usage

- **Navigation**: Use the tab buttons to switch between Home, About Us, and Contact
- **Search**: Use the search bar at the bottom to find codes
- **Ads**: Click on the advertisement panels to expand/collapse them
- **Links**: Click external links to visit featured destinations

## Customization

### Colors
Edit the color variables in `styles.css`:
- Primary green: `#2c6f4f`
- Light blue: `#7ec8e3`
- Dark text: `#333`

### Typing Text
Change the welcome message in `script.js`:
```javascript
const text = "Your custom message here!";
```

### Links
Update the external links in `index.html`:
- Advertisement links
- Contact information
- Social media links
